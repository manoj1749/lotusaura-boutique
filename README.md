# 🌸 Lotus Aura Boutique

A women's fashion e-commerce web app built with **Next.js**, **Drizzle ORM + Turso**, and **Vercel Blob** for image storage.

## Stack

- **Framework:** Next.js 16 (App Router)
- **Database:** Turso (LibSQL) via Drizzle ORM
- **Storage:** Vercel Blob
- **Styling:** Tailwind CSS v4 + Radix UI

## Environment Variables

```env
TURSO_DATABASE_URL=
TURSO_AUTH_TOKEN=
BLOB_READ_WRITE_TOKEN=
NEXT_PUBLIC_WHATSAPP_NUMBER=
ADMIN_USER=
ADMIN_PASSWORD=
```

## Dev

```bash
npm install
npm run dev
```
